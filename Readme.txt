This is a messaging application written in Java, that allows sending and receiving text messages between users.

All data are saved in a SQL database, with added password encryption for security.

Password masking has been also added for additional protection.

Users can perform CRUD action (Create, Read, Update, Delete), based on their role.

The program allows four types of users, each with added functionality.

Actions that can be performed:

-User
Send message
Receive message
Read inbox
Read outbox

-Super user
Send message
Receive message
Read inbox
Read outbox
Edit message

-Moderator
Send message
Receive message
Read inbox
Read outbox
Edit message
Delete message

-Admin
Send message
Receive message
Read inbox
Read outbox
Edit message
Delete message
Create user
Update user
Delete user
